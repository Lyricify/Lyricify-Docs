---
title: Media Session 未连接
---

请检查 Spotify 中 `使用媒体键时显示桌面叠加` 是否开启。

#### 注意
1. 只有 Lyricify 获取到了当前播放信息，才会尝试连接 Media Session！若未获取到播放信息，请先查看 [Lyricify 无法获取当前播放信息](../no-playback-info/)。
2. Media Session 仅在你使用 Spotify 桌面版 (含 Store 版) 播放时可用，其他情况 (如使用网页版、移动版等) 下不可用！
3. 系统版本低于 10.0.17763.0 时不支持 Media Session。

如果你的 Media Session 出现异常，你还可以尝试重启 Spotify 和 Lyricify。
