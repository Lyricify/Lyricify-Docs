## 应该选哪个版本

如果你不确定从哪里开始，可以先按下面的原则判断：

- 你在 Windows 上主要使用 Spotify：优先选择 [Lyricify 4](../lyricify-4/guide/)
- 你在 Windows 上使用的是支持 SMTC 的其他播放器：优先选择 [Lyricify Lite](../lyricify-lite/guide/)
- 你在 Android、iPhone、iPad、macOS 等设备上使用：优先选择 [Lyricify Mobile](../lyricify-mobile/guide/)

## 快速对比

| 版本 | 适用平台 | 主要适用场景 | 从这里开始 |
| - | - | - | - |
| Lyricify 4 | Windows | Spotify 歌词、桌面歌词、歌词管理、高级功能 | [使用指南](../lyricify-4/guide/) |
| Lyricify Lite | Windows | 其他支持 SMTC 的播放器桌面歌词 | [使用指南](../lyricify-lite/guide/) |
| Lyricify Mobile | Android / iPhone / iPad / macOS / Windows | 移动端或跨平台歌词体验 | [使用指南](../lyricify-mobile/guide/) |

## 选择建议

### 适合使用 Lyricify 4 的情况

- 你主要使用 Spotify
- 你需要更完整的桌面端功能
- 你需要歌词导入、歌词管理、曲库可用性查询等功能

### 适合使用 Lyricify Lite 的情况

- 你不主要使用 Spotify
- 你使用的播放器能提供 SMTC 信息
- 你只需要桌面歌词、灵动词岛等轻量功能

### 适合使用 Lyricify Mobile 的情况

- 你需要在手机、平板或 macOS 上使用
- 你需要 Apple 设备安装说明
- 你需要移动端的 Spotify 登录与歌词体验

## 常见情况

### 我在 Windows 上用 Spotify，该选哪个

选择 [Lyricify 4](../lyricify-4/guide/)。

### 我在 Windows 上不用 Spotify，但播放器支持系统媒体控制，该选哪个

选择 [Lyricify Lite](../lyricify-lite/guide/)。

### 我需要在 iPhone 或 iPad 上安装

选择 [Lyricify Mobile](../lyricify-mobile/guide/)，如需安装步骤可继续查看 [Apple 设备自签教程](../lyricify-mobile/ios-ipa-guide/)。

### 我主要是为了解决 Spotify 429 问题

先进入你正在使用的版本文档，再查看对应的 `自定义 Spotify API Client 配置教程`：

- [Lyricify 4](../lyricify-4/custom-api-client/)
- [Lyricify Mobile](../lyricify-mobile/custom-api-client/)
