## 产品矩阵

<CardGrid stagger>
	<Card title="Lyricify 4" icon="open-book">
		面向 Spotify 的完整桌面歌词与增强功能方案，适合 Windows 用户。
	</Card>
	<Card title="Lyricify Mobile" icon="mobile">
		面向 Android、iPhone、iPad、macOS 等平台的移动端歌词体验。
	</Card>
	<Card title="Lyricify Lite" icon="star">
		面向接入 SMTC 的 Windows 播放器，提供桌面歌词与灵动词岛等功能。
	</Card>
</CardGrid>

## 从哪里开始

- 首次使用 Lyricify 4：从[Lyricify 4 使用指南](/lyricify-4/guide/)开始。
- 需要在 iPhone 或 iPad 上安装：查看[Lyricify Mobile 使用指南](/lyricify-mobile/guide/)中的安装部分。
- 频繁遇到 `429`：查看对应产品文档中的 `自定义 Spotify API Client 配置教程` 页面。
