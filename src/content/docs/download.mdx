## 商店版说明

- Lyricify 4 和 Lyricify Lite 提供 Microsoft Store 版本
- Lyricify Mobile 当前没有 Microsoft Store 版本
- 如果您已经购买过 Lyricify 4 或 Lyricify Lite，在 Microsoft Store 版本异常时，也可以关注 [GitHub Releases](https://github.com/WXRIW/Lyricify-App/releases) 中提供的相关安装包

## 安装前说明

### Windows

- Lyricify 4 需要 `.NET Desktop Runtime 6.0`
- Lyricify Lite 需要 `.NET Desktop Runtime 8.0`

### Apple 设备

- 如需在 iPhone 或 iPad 上安装 Lyricify Mobile，请查看 [Apple 设备自签教程](/lyricify-mobile/ios-ipa-guide/)
